## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/kathan-shah1893/jarvish.git
   cd jarvish
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Run the program:

   ```bash
   python main.py
   ```

2. Follow the voice prompts and speak your command clearly.
